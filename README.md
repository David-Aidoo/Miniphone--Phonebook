# Miniphone--Phonebook
This is a console based application that is used to keep track of contacts.
Some of the major things you could do with this application includes:
1. Creating contacts: the contact may be a business contact or a personal contact.
    The business contact and personal contacts are objects which are created from different classes. What differentiates the personal contact from the business contact
    is that the business contact has the email and company name as variables with their respective set and get methods.
    This application validates the contact by checking if it matches the contact of a ghanaian contact using the regex method.It also validates the email using the        regex method.
2. Viewing the contacts.
3. Making modifications to the contacts: We make modifications to the contacts with the help of the set methods.
4. Add a contact to favourites.
5. Search for a contact
6. Remove contacts.

Some of the concepts that were considered when making this project includes: Encapsulation, OOP, Abstraction, Storage structures and others.
